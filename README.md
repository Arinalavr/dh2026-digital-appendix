# DH2026 Digital Appendix

This repository contains the public digital appendix for Arina Lavrova's DH2026 research-in-progress poster, *Reimagining an Abandoned Coastal Hotel*.

The site presents a small-data urban humanities project about an abandoned coastal hotel in Montenegro. It combines field interviews, researcher-verified qualitative coding, AI-assisted analysis support, 3D/360 documentation, and speculative spatial scenarios.

## What Is Included

- Project overview and current research stage
- Embedded 360 degree site documentation via Kuula
- AI-assisted qualitative workflow
- Preliminary Stage 1 findings
- Python visualization outputs
- Submitted DH2026 abstract
- Literature and references

## Research Ethics

Only public-facing research materials are included here. Raw interview audio, transcripts, respondent details, and identifiable field materials are not published in this repository.

## Tech

Static HTML, CSS, and JavaScript.
